Iterative Back-Translation-Style Data Augmentation for Low Resource ASR and TTS
  • Developed a novel iterative back-translation style approach to augment data for Automatic Speech Recognition (ASR) and Text to Speech (TTS) for Malayalam, a low resource Dravidian language
  • Adapted back-translation style data augmentation technique from machine translation to speech processing by leveraging speech-only data and text-only data to create a pseudo-parallel corpus for both ASR and TTS training
  • Speech-only data was fed to the ASR model trained on the limited parallel data, whose output was used to create a pseudo-parallel corpus to be used as the input to fine-tune TTS model in the next iteration and vice versa
  • Implemented a conformer-based ASR model with linear fusion of HuBERT and spectrum-based features. TTS model was a combination of Glow-TTS and Hifi-GAN
  • Achieved up to 6.91% and 10.87% reduction in Word Error Rate (WER) and Character Error Rate (CER) respectively for ASR and a 2.91% improvement in Mel-cepstral distortion (MCD) for TTS upon using just 3000 monolingual text sentences out of a possible 600,000 due to resource constraints
  • Poster here
